# ■ 機能
  - ログイン機能

  - 一覧機能
    - 投稿した料理が表示される

  - 投稿機能
    - 料理の投稿ができる

  - 編集機能
    - 投稿の編集ができる

  - 削除機能
    - 投稿を削除できる

  - 検索機能
    - 投稿の検索ができる

  - カレンダー機能
    - カレンダーに作りたい料理名などをメモできる

  - マップ機能
    - google_mapが利用できる
  
  - 料理動画機能
    - 料理動画の検索および視聴ができる
